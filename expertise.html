Software Platforms

Modelling Platforms

We maintain expertise across all major power system simulation platforms.

DIgSILENT PowerFactory

Industry-leading power system analysis software for transmission, distribution, and industrial network modelling.

  • Load flow & fault analysis
  • Dynamic & RMS simulation
  • Protection coordination
  • Power quality studies
  • Scripting automation (Python/DPL)

PSCAD

Electromagnetic transient simulation for detailed converter modelling and controller design validation.

  • HVDC & FACTS modelling
  • Wind turbine converter control
  • Custom controller implementation
  • Time-domain electromagnetic transients
  • Hardware-in-the-loop preparation

PSS®E

Transmission planning and operations software for large-scale network analysis and stability studies.

  • Transmission planning studies
  • Stability & transient analysis
  • Interconnection studies
  • Network equivalent modelling
  • PSS®E Python API automation

CDEGS

Specialist earthing and electromagnetic field analysis for substations and pipeline interference studies.

  • Substation earthing design
  • Soil resistivity analysis
  • Pipeline interference assessment
  • EMF environmental studies
  • Cable sheath current analysis
